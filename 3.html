SELECT

The SELECT statement is used to select data from a database.

SELECT column1, column2, ...
FROM table_name;

SELECT DISTICT

The SELECT DISTINCT statement is used to return only distinct (different) values.

SELECT DISTINCT column1, column2, ...
FROM table_name;

SELECT ALL columns

The SELECT statement is used to select data from a database.

SELECT * FROM table_name;

     
FIRST SELECT FIRST (columnName) FROM tableName The SELECT FIRST clause is used in some SQL databases (primarily MS Access) to retrieve the first record from a table based on the order in which data is stored or queried
LAST SELECT LAST (columnName) FROM tableName SELECT LAST is a concept or function often used to describe retrieving the last record or last row from a table in SQL.
TOP SELECT column1, column2, … TOP count FROM tableName The SELECT TOP clause in SQL only returns the specified number of rows from the table.
Limit SELECT columnName FROM tableName limit 5 SQL LIMIT Clause limits the number of results returned in the results set.
FETCH FIRST SELECT columns FROM table WHERE condition FETCH FIRST n ROWS ONLY; SQL FETCH FIRST clause fetches the first given number of rows from the table..
Date SELECT FirstName, LastName, InvoiceDate FROM Employee WHERE InvoiceDate BETWEEN '2021-01-01' AND '2022-01-01'; SQL functions allow you to extract specific components (like year, month, or day) from a DATE or DATETIME column.

YEAR(): Extracts the year from a date.
MONTH(): Extracts the month from a date.
DAY(): Extracts the day from a date.


INSERT Statement

The INSERT statement is used to add new records to a database table.

INSERT INTO Employees (EmployeeID, EmployeeName, Age, Department)
VALUES
(1, 'John Doe', 30, 'Engineering'),
(2, 'Jane Smith', 28, 'Marketing'),
(3, 'Sam Brown', 35, 'Sales'),
(4, 'Lucy Green', 25, 'Human Resources');



INSERT INTO Employees (EmployeeID, EmployeeName, Age, Department)
SELECT EmployeeID, EmployeeName, Age, Department
FROM NewEmployees
WHERE Age > 30;

Delete Statement

The SQL DELETE statement removes one or more rows from a database table based on a condition.

Delete From table_name
where condition
